Feature extractor class for Netprob2vec Convert graph probability matrix to a document specify extractor for ordered/unordered/rounded/tagged/untagged

ordered_probability_extractor(self)¶

Ordered sequence (decreasing probability) of nodes or bin index (of distances) for each rooted node converted to words This is for networks edge-weighted with real data like gene(perhaps not for weights with stat data like correlation or p-values - To be tested).

prob_to_string_feature_extractor(self)¶

Node-wise probability values for each rooted node converted to words Does not really work, does not create high frequency words, so no vocabulary created.

unordered_probability_extractor(self)¶

Unorodered sequence of nodes (non-neg probabilities) or bin index (of distances or node labels) for each rooted node converted to words. For LFR, unweighted synthetic data.
